The dataset used for this project are the Netflix prize data set\cite{dataset}. It consists of \path{movie_titles.txt} which lists all movies on the format given below.
\begin{verbatim}
  [MovieID],[YearOfRelease],[Title]
  1,2003,Dinosaur Planet
  2,2004,Isle of Man TT 2004 Review
  3,1997,Character
  .
  .
\end{verbatim}
\noindent
In addition, there are a .txt file for every movie in the \path{training_set} folder that contains information about what rating a given user has given the movie, and on what date the movie was rated. It is formatted as following
\begin{verbatim}
  [MovieID]
  [CustomerID],[Rating],[Date]
  1:
  1488844,3,2005-09-06
\end{verbatim}
